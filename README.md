## Installation

    $ npx create-react-app
    $ npm react-router-dom
    $ npm i firebase
    $ npm install react-icons --save
    $ npm i react-photo-view
    $ npm firebase deploy
    $ npm install -D tailwindcss
    $ npm install --save react-toastify
